# Mobile Client Service

## Prerequisites

* Golang (1.10)
* Nodejs

## Setup

```bash
make setup
```

## Build

```bash
# Build the API server
make build
# Build the UI
make ui
```

## Run

```bash
make serve
```

## Test

```bash
make test
```
